## ğŸš€ Getting Started (5 Minutes)

### 1. Start Backend
```powershell
cd backend
npm start
```
**Runs on:** http://localhost:3001

### 2. Start Frontend
```powershell
cd frontend
npm run dev
```
**Runs on:** http://localhost:3000

### 3. Create a Workflow
1. Open http://localhost:3000
2. Enter: "Summarize tech news daily"
3. Click "Generate Workflow"
4. Click "Run"

**Done!** You've just created and executed an AI-generated workflow.
